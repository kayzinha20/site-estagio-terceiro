## Formulário de Login com Bootstrap

## Sobre o projeto

Este projeto consiste em um formulário de login responsivo, desenvolvido utilizando Bootstrap, com foco em praticar a criação de interfaces modernas, acessíveis e adaptáveis a diferentes tamanhos de tela.

O layout utiliza componentes prontos do Bootstrap, garantindo rapidez no desenvolvimento e padronização visual.

## Tecnologias utilizadas

HTML5

CSS3

Bootstrap 5

## Funcionalidades

Layout responsivo

Tema escuro utilizando data-bs-theme="dark"

Campos de login com efeito floating label

Checkbox “Remember me”

Botão de login padrão

Botão alternativo para login com Google

## Objetivo

Projeto desenvolvido para aprendizado e prática do framework Bootstrap, reforçando conceitos como:

Uso de classes utilitárias

Componentes prontos do Bootstrap

Responsividade

Organização de layout com Flexbox

## Anotações

Bootstrap é um framework front-end (HTML, CSS e JavaScript) com o objetivo de facilitar a criação de interfaces web responsivas, oferecendo componentes prontos e estilos padronizados.
